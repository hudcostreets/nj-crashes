# `raw/` — primary-source archive

A frozen mirror of the original bulk-data files behind NJ Crash Dashboard
([crashes.hudcostreets.org](https://crashes.hudcostreets.org)). Useful for
researchers, journalists, and anyone wanting to reconstruct the analyses
themselves.

## What's here

- [`njdot/data/`](njdot/data/) — **NJ DOT bulk crash dumps**, 2001–2025.
  Three format eras:
  - **2001–2022** (per-table, statewide): five tables per year —
    Accidents, Drivers, Occupants, Pedestrians, Vehicles — that JOIN on
    a crash case key.
  - **2023** (per-table, per-county + statewide): same five tables,
    split per-county (21 × 5 zips) alongside the statewide set.
  - **2024+** (denormalized CSV only): a single `Crash.csv` per year
    (~700 MB) generated by NJ DOT's AASHTO BTDS dashboard, with all
    driver/vehicle/occupant/pedestrian facts collapsed into wide
    duplicated columns. The clean PK-based JOIN model is gone.
- [`njdot/data/fields/`](njdot/data/fields/) — column schemas (JSON +
  original NJDOT PDF data dictionaries) for the fixed-width `.txt` files
  inside each `.zip` (per-table era only).
- [`njsp/data/annual-summaries/`](njsp/data/annual-summaries/) — **NJ
  State Police annual fatal-crash report PDFs**, 2010–2025. Two reports
  per year (`ptccr_YY.pdf`, `swfcs2_YY.pdf`); used to backfill the
  pre-2008 NJSP data the live XML feed doesn't cover.

## Why we mirror this

Restoring the per-table layout for 2024+ — so analyses of pedestrians,
occupants, and vehicles can again be modelled on their natural
primary keys — is the motivating ask behind this archive.

## Format conventions

Files end in:
- `.zip` — original per-table archive as published by NJ DOT (2001–2023).
  Contains a single fixed-width `.txt` with the same stem.
- `.pqt` — Parquet copy parsed via the `fields/` schemas. Same data,
  range-paginated, much faster to navigate.
- `.csv` — denormalized AASHTO export (2022+). 2024 and 2025 only have
  this format.
- `.pdf` — NJ State Police annual reports.

Click any `.zip` to inspect its entry list; click an entry to preview
it. Click a `.pqt` for a paginated table view. Click a directory to
list its contents.
