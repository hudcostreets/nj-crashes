# `njdot/data/2025/` — denormalized `Crash.csv` only

Same format as [2024](../2024/) — a single denormalized
[`Crash.csv`](Crash.csv) (~700 MB) generated by the AASHTO dashboard.

The per-table bulk dumps that NJ DOT published through 2023 (see
[`../2022/`](../2022/) and [`../2023/`](../2023/)) are no longer
distributed. We are working with NJ DOT (DOT-BTDS) to restore the
per-table layout for 2024+; see [`../2024/`](../2024/) for the rationale.
